UX 状态流

1进入消息页后，优先展示搜索、归档入口和会话列表，底部 Tab 固定。
2左滑会话，露出置顶、免打扰、删除；右滑会话，露出归档。
3进入选择模式后，导航切换为取消、标题、全选；每行出现选择圆点。
4底部操作栏根据选择数量显示已读、归档、删除，并实时更新“已选 N 项”。

开发标注

  • 画布iPhone 393 × 852，可适配安全区。
  • 列表行高默认 74px，头像 50px，左边距 16px。
  • 搜索框高度 40px，圆角 8px，背景 #F0F3F8。
  • 分隔线#E6EAF0，左侧从文本区开始对齐。
  • 未读徽标最小 20 × 20，红色 #FF3B30。
  • 主色#246BFE，用于选中、操作和重点状态。
