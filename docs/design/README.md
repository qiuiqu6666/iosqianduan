# RainbowChat 新版设计稿

本目录放当前 iOS 项目的新版 UI 设计探索稿。

## 文件

- `rainbowchat-redesign.html`：可直接用浏览器打开的设计看板，包含 5 张移动端关键界面。
- `message-ui-ux-design.html`：基于 `export.psd` 整理的消息页面 UI/UX 设计图，包含默认态、选择态、交互流和开发标注。
- `export-html/index.html`：`export.psd` 的图片式 HTML 还原版。

## 覆盖页面

1. 登录页
2. 消息列表
3. 聊天页
4. 通讯录
5. 钱包 / 我的入口

## 设计方向

- 消息优先：消息列表提高信息密度，突出未读、@我、置顶和时间。
- 工具感更强：减少过度装饰，用稳定的白底、浅灰分隔和蓝色品牌色承载状态。
- 保留业务入口：钱包、红包、转账、群聊、文件等能力继续可见，但不压过聊天主任务。
- 可落地：设计尽量贴近当前 UIKit + XIB 工程已有能力，图标优先复用 `Images.xcassets` 中的现有资源。

## 建议落地顺序

1. 先重做登录页和底部 Tab 视觉。
2. 再重做消息列表 cell、搜索框、筛选切换。
3. 最后处理聊天页输入栏、引用消息、语音气泡和钱包页。
